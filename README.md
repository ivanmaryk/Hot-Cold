❄️ Hot&Cold – Угадай число с подсказками

**Интеллектуальная игра "Горячо-холодно"** на **7 языках программирования**.  
Компьютер загадывает число, а вы пытаетесь его отгадать. Каждая попытка сопровождается подсказкой:  
`🔥 ГОРЯЧО` – вы близко, `❄️ ХОЛОДНО` – вы далеко, а также стрелками направления `▲` (число больше) или `▼` (число меньше).

---

## ✨ Уникальные особенности (общие для всех версий)

- 🎯 **Динамические подсказки** – чем ближе вы к числу, тем "горячее" становится.
- 📈 **История попыток** – сохраняются все введённые числа с пометкой "горячо/холодно".
- 🏆 **Подсчёт очков** – минимальное количество попыток запоминается как рекорд.
- 🎚️ **Три уровня сложности**:
  - *Лёгкий* – число от 1 до 50 (бесконечные попытки)
  - *Средний* – от 1 до 100 (максимум 10 попыток)
  - *Сложный* – от 1 до 200 (максимум 7 попыток)
- 💾 **Сохранение рекордов** – лучший результат сохраняется в файл и загружается при старте.
- 🎨 **Цветной вывод** – интуитивно понятные цвета (красный – горячо, синий – холодно, зелёный – победа).
- 🕹️ **Интерактивное меню** и возможность играть несколько раундов без перезапуска.
- 🤖 **Режим "Автоугадывание"** (для демонстрации) – компьютер сам угадывает число, показывая свою логику.

## 🚀 Как запустить?

| Язык       | Файл            | Команда запуска                              |
|------------|-----------------|----------------------------------------------|
| Python     | `hotcold.py`    | `python hotcold.py`                          |
| Go         | `hotcold.go`    | `go run hotcold.go`                          |
| C++        | `hotcold.cpp`   | `g++ -std=c++17 hotcold.cpp -o hot && ./hot` |
| JavaScript | `hotcold.js`    | `node hotcold.js`                            |
| Rust       | `hotcold.rs`    | `cargo run` (или `rustc hotcold.rs && ./hotcold`) |
| Java       | `HotCold.java`  | `javac HotCold.java && java HotCold`         |
| C#         | `hotcold.cs`    | `dotnet script hotcold.cs` или `csc hotcold.cs && mono hotcold.exe` |

Все версии используют только стандартные библиотеки (кроме Rust, где нужны `rand` и `serde_json` – укажем в `Cargo.toml`).

## 📦 Установка зависимостей (если нужны)

- **Python**: стандартная библиотека + `colorama` (для цвета) – установка: `pip install colorama`
- **JavaScript**: только Node.js, используется `readline` и `fs`
- **Java**: только JDK
- **C++**: только компилятор с поддержкой C++17
- **C#**: .NET SDK
- **Go**: только Go
- **Rust**: добавьте в `Cargo.toml`:
  ```toml
  [dependencies]
  rand = "0.8"
  serde_json = "1.0"
🛠️ Продвинутые фичи по языкам
Python: графический прогресс-бар попыток, аргументы командной строки для автоматической игры.

Go: горутины для асинхронного таймера (подсказка времени).

C++: использование std::variant для хранения разных типов подсказок, умные указатели.

JavaScript: асинхронное меню с async/await, красивые emoji-подсказки.

Rust: строгая обработка ошибок через Result, итераторы для истории.

Java: паттерн State для управления состояниями игры, лямбды для обработки ввода.

C#: LINQ для анализа истории, события на каждую попытку.
